# 📦 Blender - Guia Rápido (Anotações Pessoais)

> Repositório para estudo e consulta rápida de atalhos e conceitos do Blender.

---

## 🧠 Conceitos Básicos

### 🧱 Elementos de um objeto 3D

- **Vértice (Vertex)** → ponto  
- **Aresta (Edge)** → linha entre dois pontos  
- **Face (Face)** → superfície  

---

### 🌐 Eixos no Blender

- **X (vermelho)** → esquerda/direita  
- **Y (verde)** → frente/trás  
- **Z (azul)** → cima/baixo  

💡 Tudo no Blender gira em torno desses eixos.

---

## 🎥 Navegação (Câmera)

- **Scroll pressionado (clique)** → orbitar  
- **Scroll (rolar)** → zoom  
- **Shift + Scroll pressionado** → pan (movimento lateral)  

💡 Dica importante:  
Ative **"Zoom to Mouse Position"** para uma navegação mais precisa.

---

## 🖱️ Seleção e Modos

### 🔄 Alternar modo

- `Tab` → alterna entre:
  - Modo Objeto
  - Modo Edição

---

### ✏️ No modo edição

- `1` → Vértices  
- `2` → Arestas  
- `3` → Faces  

⚠️ Use os números **acima do teclado**, não o numérico.

---

### 🎯 Seleção rápida

- `L` → seleciona parte inteira conectada  
- `X` → deletar seleção  

---

## ⚙️ Comandos Básicos

- `Enter` → confirmar  
- `Esc` → cancelar  
- `Ctrl + Z` → desfazer  

---

## 🧩 Modelagem (Essencial)

### ➕ Criação

- `Shift + A` → adicionar objeto  

---

### 🔄 Transformações

- `G` → mover  
- `R` → rotacionar  
- `S` → escalar  

💡 **Travar no eixo:**

bash
G → X
R → Z
S → Y

---
### 💡 Resetar valor
- G → X → 0
---

### ✂️ Edição de malha
E → extrudar
Ctrl + R → loop cut
K → knife (corte manual)
M → merge (unir vértices)

---

### 🔧 Modificações úteis
Ctrl + B → bevel (arredondar)
Alt + E → opções de extrusão
Shift + D → duplicar
Ctrl + J → juntar objetos

---
### 🎯 Extras importantes
O → edição proporcional
U → UV Mapping

---
### 🚀 Dicas de Ouro
Pense em forma + simplicidade (low poly primeiro)
Use loop cut (Ctrl + R) ao invés de adicionar vértices aleatórios
Trabalhe com poucos vértices → mais controle
Domine bem:
Extrude
Scale
Loop Cut
